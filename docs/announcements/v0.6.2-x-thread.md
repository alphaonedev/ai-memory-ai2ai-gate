**5 / 12**

The centerpiece bug we closed this cycle: S40 bulk-write fanout.

500 memories posted in one /bulk, quorum met, HTTP 200 — but one peer would end up with 499.

Root cause: post-quorum detached POST was fire-and-forget. Fire silently, forget silently, corrupt "shared memory" silently.
